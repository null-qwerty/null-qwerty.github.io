## 列表

### 无序表

```
* 无序
* 无序
* 无序
```

等同于

```
+ 无序
+ 无序
+ 无序
```

等同于

```
- 无序
- 无序
- 无序
```

* 无序
* 无序
* 无序

### 有序表

```
1. 有序
2. 有序
3. 有序
```

1. 有序
2. 有序
3. 有序

### 复选框

```
- [ ] 未选
- [x] 已选
```

- [ ] 未选
- [X] 已选
